# Program Administrasi Perpustakaan Bahagia

Project ini merupakan pembuatan aplikasi CRUD sederhana
di pyton dengan studi kasus administarasi perpustakaan. Dengan program ini, user dapat melakukan aksi Read, 
Create, Edit, dan Delete terhadap dataset yang tersimpan
di dalam program. Program perpustakan ini dapat dijalankan dengan dua 
sudut pandang, yaitu sebagai pengunjung dan sebagai
admin perpustakaan.

**CRUD Administrasi Perpustakaan**

Read:

• User dapat melihat semua data buku yang tersimpan di dalam program.

• User dapatmelihat anggota perpustakaan yang terdaftar.

Create:

• User dapat menabahkan buku baru ke dalam dataset  apabila buku tersebut memang belum tersinpan didalam dataset.

• User dapat mendaftar sebagai anggota baru perpustakaan

Edit:

• Saat user meminjam buku, secara otomatis status  buku yang  dipinjam akan berubah menjadi “Tidak tersedia” di dalam dataset.

• Saat buku dikembalikan status buku akan berubah menjadi “Tersedia”.

Delete:

• User dapat menghapus data buku yang tersinpan dengan memasukkan index buku tersebut.

Filtering:

• Saat use akan meminjam buku, user dapat melihat semua data buku yang dapat dipinjam hanya buku dengan status buku “Tersedia”.
